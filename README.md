### If Statements

`if_statements.py` demonstrates the usage of if, elif, and else blocks.

```python
# Example of if-elif-else
x = 10
if x > 10:
    print("x is greater than 10")
elif x == 10:
    print("x is exactly 10")
else:
    print("x is less than 10")
